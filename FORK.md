# GNU Backgammon - macOS Fork

This is a fork of GNU Backgammon (gnubg) 1.08.003 with updated build system for macOS.

## About This Fork

**Upstream Project:** GNU Backgammon
**Upstream Version:** 1.08.003
**Upstream Repository:** https://www.gnu.org/software/gnubg/
**Fork Maintainer:** kenr
**Fork Purpose:** Updated build system for modern macOS compatibility

## Why This Fork Exists

The original GNU Backgammon project is rarely updated, and the build system needed modifications to work properly on modern macOS systems. Rather than waiting for upstream changes, this fork maintains macOS-specific build improvements.

## Key Modifications

- Updated build system for macOS compatibility
- Build configuration changes for modern macOS toolchains
- macOS-specific fixes and patches

See `CHANGES.md` for a detailed list of all modifications from the upstream version.

## Upstream Tracking

Since GNU Backgammon is rarely updated, this fork:
- Does not automatically sync with upstream
- Manually reviews upstream releases when they occur
- Applies relevant upstream changes as needed
